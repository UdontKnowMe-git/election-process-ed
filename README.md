# 🗳️ Democracy Now: An Interactive Election Journey

**Democracy Now** is an immersive, AI-powered educational platform designed to demystify the election process. [cite_start]Built for the **Virtual: Prompt Wars** hackathon on Hack2skill, this application replaces traditional, "textbook-style" learning with a highly interactive, gamified experience[cite: 17, 20].

## 🌟 Key Features

* [cite_start]**Dual-Perspective Timelines**: Explore the election cycle through a smooth 2D vertical timeline or an experimental 3D "fly-through" experience[cite: 38, 97].
* [cite_start]**Voter's Quest Quiz**: A fast-paced, Kahoot-style challenge with point multipliers, streaks, and real-time feedback to test your civic knowledge[cite: 40, 98].
* [cite_start]**Civic Guru AI Assistant**: An intelligent chatbot powered by **Gemini 2.5** that provides non-partisan, context-aware answers to complex election questions[cite: 30, 99].
* [cite_start]**Myth vs. Fact Swipe**: A "Tinder-style" card interface for quickly debunking common election misconceptions[cite: 27, 98].
* [cite_start]**Butterfly Effect Visualizer**: An interactive tool that illustrates the long-term impact of local policy votes on a community[cite: 59, 60].

## 🛠️ Tech Stack

[cite_start]This project leverages a modern, high-performance stack optimized for "vibe coding" and efficiency[cite: 5, 57]:

* [cite_start]**Frontend**: React + Vite [cite: 53, 56]

## 🚀 Getting Started

### Prerequisites
* Node.js (v18 or higher)
* [cite_start]A Google AI Studio API Key (for the Civic Guru assistant) [cite: 99]

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/UdontKnowMe-git/election-process-ed.git
    cd election-process-ed
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Set up environment variables**:
    Create a `.env` file in the root directory and add your Gemini API key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```
4.  **Run the development server**:
    ```bash
    npm run dev
    ```
